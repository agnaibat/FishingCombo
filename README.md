# Рыбалка В Сборе — лендинг

Готовый одностраничный сайт на чистом HTML/CSS/JS, без сборки и зависимостей (кроме шрифтов Google Fonts, подключённых по CDN).

## Как открыть
Просто откройте `index.html` в браузере, либо запустите локальный сервер:
```
python3 -m http.server 8080
```
и перейдите на `http://localhost:8080`.

## Структура
```
index.html   — разметка всех секций
styles.css   — стили, цветовая схема, анимации, адаптив
script.js    — шапка при скролле, мобильное меню, скролл-анимации, аккордеон FAQ, лайтбокс галереи
favicon.svg  — иконка сайта
images/      — реальные фотографии комплекта
```

## Изображения
В hero, в блоке «Что входит в комплект» и в 4 из 6 карточек галереи уже стоят реальные фотографии из `images/`:
- `rod-tip-lure.png` — hero и карточка «Общий комплект»
- `rod-full.png` — блок «Что входит в комплект»
- `reel.png` — карточка «Катушка»
- `rod-guide.png` — карточка «Кольца»
- `line-spool.png` — карточка «Плетёный шнур»

Карточки **«Рукоять»** и **«Процесс рыбалки»** пока оставлены как line-art SVG-иллюстрации — подходящих фото под них не было в присланном наборе. Как только появятся фото рукояти крупным планом и/или кадр с водоёма, пришлите — заменю на реальные снимки таким же образом.

## Масштабирование до интернет-магазина
Секции `#kit`, `#specs` и `.gallery-item` уже структурированы как переиспользуемые карточки — при переходе на каталог их разметку и стили можно использовать как шаблон карточки товара. Цветовые токены и типографика вынесены в `:root` в начале `styles.css`, поэтому фирменный стиль масштабируется без переписывания CSS.
